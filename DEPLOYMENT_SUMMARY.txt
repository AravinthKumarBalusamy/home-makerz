├── Cost: Free (with limitations)
├── Steps:

│   2. Get Anthropic API key (optional)
│   3. Deploy to Render.com

REQUIRED ENVIRONMENT VARIABLES
================================================================================

✅ NODE_ENV=production
✅ APP_PIN=your-pin-here (CHANGE THIS!)
✅ MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/homemakerz
⚠️  ANTHROPIC_API_KEY=sk-ant-xxx (optional, for bill scanner)

================================================================================
DEPLOYMENT STEPS (QUICK VERSION)
================================================================================

1. MongoDB Atlas Setup (3 min)
   → Sign up at https://www.mongodb.com/cloud/atlas
   → Create free M0 cluster
   → Get connection string
   → Whitelist all IPs (0.0.0.0/0)

2. Anthropic API Key (2 min) - OPTIONAL
   → Sign up at https://console.anthropic.com/
   → Create API key
   → Copy key (starts with sk-ant-)

3. Deploy to Render (5 min)
   → Sign up at https://render.com with GitHub
   → Connect repository
   → Add environment variables
   → Deploy!

4. First Login
   → Visit your Render URL
   → Enter your PIN
   → Start using HomeMakerz!

FREE TIER LIMITATIONS
================================================================================

Render.com Free Tier:
- App sleeps after 15 minutes of inactivity
- First request after sleep takes ~30 seconds to wake up
- 750 hours/month (sufficient for personal use)

MongoDB Atlas Free Tier:
- 512MB storage
- Shared cluster
- Sufficient for personal use

Anthropic API:
- Pay-as-you-go
- ~$0.25 per 1M tokens
- Estimated $1-5/month for moderate bill scanning

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Cannot connect to database"
Fix: Check MongoDB connection string and IP whitelist

Issue: "Invalid PIN"
Fix: Verify APP_PIN environment variable in Render

Issue: "Bill scanner not working"
Fix: Verify ANTHROPIC_API_KEY is set correctly

Issue: "App is slow"
Fix: Free tier sleeps - upgrade to paid tier or use uptime monitor

================================================================================
SECURITY RECOMMENDATIONS
================================================================================

✅ Use a strong PIN (at least 6 digits)
✅ Keep MongoDB password secure
✅ Don't share Anthropic API key
✅ Enable HTTPS (automatic on Render)
✅ Create regular backups
✅ Whitelist specific IPs in MongoDB (optional)
