# Deployment Checklist

## Pre-Deployment

- [ ] Code pushed to GitHub: https://github.com/AravinthKumarBalusamy/home-makerz.git
- [ ] All dependencies listed in package.json
- [ ] .env.example file present with all required variables
- [ ] Dockerfile present and tested (optional)

## MongoDB Setup

- [ ] MongoDB Atlas account created
- [ ] Free cluster (M0) created
- [ ] Database user created with password
- [ ] Network access configured (0.0.0.0/0 for all IPs)
- [ ] Connection string copied
- [ ] Connection string tested locally

## API Keys

- [ ] Anthropic API key obtained (optional for bill scanner)
- [ ] API key tested with a sample request
- [ ] API key has sufficient credits/quota

## Render.com Deployment

- [ ] Render account created
- [ ] GitHub repository connected
- [ ] Web service created
- [ ] Environment variables configured:
  - [ ] NODE_ENV=production
  - [ ] APP_PIN=your-pin
  - [ ] MONGODB_URI=your-connection-string
  - [ ] ANTHROPIC_API_KEY=your-key (optional)
- [ ] Build completed successfully
- [ ] Service is live and running

## Post-Deployment Testing

- [ ] App URL accessible
- [ ] Login page loads
- [ ] Can login with PIN
- [ ] Dashboard loads with default data
- [ ] Can create a new expense
- [ ] Can create a new task
- [ ] Settings page accessible
- [ ] Bill scanner page loads
- [ ] Can upload and scan a bill (if API key configured)
- [ ] Backup feature works
- [ ] All navigation links work

## Configuration

- [ ] Updated salaries in Settings
- [ ] Configured recurring expenses
- [ ] Customized dashboard widgets
- [ ] Set up expense categories (if needed)

## Security

- [ ] Changed default PIN to a strong one
- [ ] Verified session management works
- [ ] Tested logout functionality
- [ ] Confirmed unauthorized access is blocked

## Backup & Recovery

- [ ] Created first backup
- [ ] Downloaded backup file
- [ ] Stored backup securely
- [ ] Tested restore functionality

## Documentation

- [ ] README.md updated with deployment URL
- [ ] Team members have access credentials
- [ ] Backup schedule documented

## Monitoring

- [ ] Bookmarked Render dashboard
- [ ] Bookmarked MongoDB Atlas dashboard
- [ ] Set up uptime monitoring (optional)
- [ ] Configured error alerts (optional)

## Optional Enhancements

- [ ] Custom domain configured
- [ ] SSL certificate verified
- [ ] PWA installed on mobile device
- [ ] Upgraded to paid tier (if needed)
